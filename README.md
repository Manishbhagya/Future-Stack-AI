# Future Stack AI

Production AI services platform — Draftr SaaS-inspired design.

## Tech Stack

- Next.js 16 + React 19 (App Router)
- Clerk (authentication)
- Supabase (PostgreSQL)
- Framer Motion (scroll animations)
- Outfit + DM Sans (typography)
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Routes

| Route | Page |
|---|---|
| `/` | Homepage (hero, marquee, features, process, integrations, testimonials, pricing, CTA) |
| `/services/[slug]` | Service details (chatbots, ML, web dev, data analytics, cloud, automation) |
| `/about` | About us |
| `/contact` | Contact form |
| `/blog` | Blog index |
| `/blog/[slug]` | Blog post |
| `/pricing` | Pricing plans |
| `/faqs` | FAQ page |
| `/waitlist` | Waitlist signup |
| `/changelog` | Changelog |
| `/privacy-policy` | Privacy policy |
| `/power-ups` | Power-ups directory |
| `/dashboard` | User dashboard (auth required) |
| Auth routes | Sign-in, sign-up (Clerk) |

## Design

Inspired by the Draftr SaaS template — light theme with dark hero/footer, gold accent (`#D4A853`), pill badges, arrow buttons, card-based layouts with hover borders. All sections animated on scroll via Framer Motion.
