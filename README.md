# Future Stack AI

Production AI services platform.

## Tech Stack

- Next.js 16 + React 19
- Clerk (authentication)
- Supabase (PostgreSQL)
- Deployed on Vercel

## Getting Started

1. Clone the repo
2. Copy `.env.example` to `.env.local` and fill in values
3. Run `npm install`
4. Run `npm run dev`

Open [http://localhost:3000](http://localhost:3000) to view the site.
